Detailed Description

Implements an end-coupled resonator bandpass filter.

This class synthesizes bandpass filters using end-coupled resonator topology, where resonators are coupled through series capacitors at their ends. Reference: "Microwave Engineering", David M. Pozar. 4th edition. Page 441-442

Note
Supports:
  • Ideal transmission lines
  • Microstrip transmission lines
