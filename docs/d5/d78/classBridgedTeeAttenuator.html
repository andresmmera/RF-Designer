Detailed Description

Bridged-Tee attenuator implementation.

Four-resistor topology: R1 (series), R2/R3 (shunt to ground = Z0), R4 (bridge to ground) Reference: RF Design Guide. Systems, Circuits, and Equations. Peter Vizmuller. Artech House, 1995
