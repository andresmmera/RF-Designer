Detailed Description

Directive for creating a figure with Markdown compatible syntax.

Example::

    :::{figure-md} target
    <img src="img/fun-fish.png" alt="fishy" class="bg-primary mb-1" width="200px">

    This is a caption in **Markdown**
    :::
