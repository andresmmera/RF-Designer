Detailed Description

Implements synthesis of various direct-coupled resonator filter topologies.

This class provides comprehensive design equations for direct-coupled bandpass filters

Reference: "Microwave Filters, Impedance-Matching Networks, and Coupling Structures" by Matthaei, Young, and Jones.
